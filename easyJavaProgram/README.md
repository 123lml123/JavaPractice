# easyJavaProgram
简单Java算法程序

## 1. squarepattern
给定整数n，以下列方式打印n行：
```
1*2*3*4
9*10*11*12
13*14*15*16
5*6*7*8
```

## 2. GenPattern
生成图案：
```
    *
   * *
  * * *
 * * * *
```
其中，控制台输入的数值控制图案生成的行数

## 3. TestPrimeNumber
生成所有的素数

## 4. ArraySortQuantity
对输入的数组进行排序，排序规则：
按数组中元素出现的次数进行排序，出现次数多的元素排在前面

- .ArraySortQuantity: 使用JDK7及以下语法
- .ArraySortQuantityLambda: 使用JDK8语法
